# Usage

`addSbtPlugin("nl.anchormen.sbt" %% "sbt-web3j-generator" % "0.1")`

# Dependencies

```
libraryDependencies += "org.web3j" % "abi" % "3.2.0"
libraryDependencies += "org.web3j" % "core" % "3.2.0"
libraryDependencies += "org.web3j" % "crypto" % "3.2.0"
```
